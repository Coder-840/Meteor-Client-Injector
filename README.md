# Meteor-Client-Injector

Single-file injector:

- Open `injector.html` in your browser.
- Upload `ChunkyClient.zip`.
- Click **Inject + Download HTML**.
- It auto-downloads `*.meteor.injected.html`.

No Python required.


Injected client now includes a built-in click GUI toggle on **Right Shift** for module/category toggles and JSON config editing.
